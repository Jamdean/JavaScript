<h3> 내부 스크립트 외부로 분리하기 </h3>
        <script src="JS 파일 경로"></script>
         
         
        내부 스크립트를 외부로 분리하려면 따로 파일을 만들어서 저장해야 한다.
        즉, HTML 파일과 JS(자바 스크립트) 파일을 분리하여 HTML 내에서 JS파일을 연동하면 된다.
