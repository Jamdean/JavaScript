<h3> 변수란?
  변하는 데이터 값을 저장할 수 있는 메모리 공간.

  <h4> 변수 선언
    var 키워드를 변수명 앞에 붙이고 사용한다.<br>
    변수명에는 한글을 사용할 수 없으며, 영문과 숫자 일부 특수문자 (_,$)만 포함할 수 있다.<br>
   
     
    
    기본형: var 변수명; 또는 var 변수명=값;
   
    var box;
    box=100;
